# Викторина для GitHub Pages + Google Sheets

В комплекте:
- `index.html` — страница викторины
- `styles.css` — стили
- `app.js` — логика викторины
- `config.js` — сюда вставляется URL веб-приложения Google Apps Script
- `Q.txt` — файл с вопросами
- `apps-script.gs` — серверная часть для записи имени и баллов в Google Sheets

## Что обновлено для телефонов
- адаптивная вёрстка для мобильных экранов
- крупные кнопки и поля ввода для касания пальцем
- корректная работа на узких экранах без горизонтальной прокрутки
- safe area для iPhone и других телефонов с вырезами
- высота интерфейса через `100dvh`, чтобы страница не ломалась из-за мобильной адресной строки
- кнопка перехода к следующему вопросу на телефонах растягивается на всю ширину и остаётся удобной

## Как это работает
1. Пользователь вводит имя.
2. Имя сразу уходит в Google Sheets и записывается в новую строку.
3. Сайт выбирает 10 случайных вопросов из `Q.txt` без повторений.
4. После окончания викторины сайт отправляет балл в соседний столбец той же строки.

## Формат Q.txt
Каждый вопрос — отдельный блок, разделённый пустой строкой.

Пример:

ВОПРОС1
Текст вопроса.
Вариант 1
Вариант 2*
Вариант 3
Вариант 4

- Первая строка блока может быть служебной (`ВОПРОС1`) и на экран не выводится.
- Вторая строка — текст вопроса.
- Дальше идут варианты ответа.
- У правильного варианта в конце стоит `*`.
- На экране звёздочка не показывается.

## Настройка Google Apps Script
1. Откройте таблицу Google Sheets.
2. Откройте `Расширения` → `Apps Script`.
3. Вставьте код из `apps-script.gs`.
4. Сохраните проект.
5. Нажмите `Deploy` → `New deployment`.
6. Выберите тип `Web app`.
7. `Execute as`: **Me**.
8. `Who has access`: **Anyone**.
9. Нажмите `Deploy`.
10. Скопируйте URL веб-приложения.

## Настройка фронтенда
1. Откройте файл `config.js`.
2. Вставьте URL веб-приложения вместо `PASTE_YOUR_APPS_SCRIPT_WEB_APP_URL_HERE`.

## Публикация на GitHub Pages
1. Создайте репозиторий.
2. Загрузите в него файлы:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `config.js`
   - `Q.txt`
3. Включите GitHub Pages для ветки `main`.
4. Откройте опубликованный адрес сайта на компьютере и телефоне.

## Важно
- Таблицу **не нужно** оставлять публично открытой на редактирование всем подряд. После настройки Apps Script лучше оставить доступ только себе или нужным редакторам.
- Если хотите писать не в первый лист, а в конкретный, впишите имя листа в `SHEET_NAME` внутри `apps-script.gs`.
- При большом числе одновременных участников Apps Script использует блокировку, чтобы записи не налезали друг на друга.
